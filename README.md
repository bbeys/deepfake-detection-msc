# deepfake-detection-msc
MSc Computing thesis — University of Roehampton 2026. Evaluating the reliability of deepfake detection models (XceptionNet, EfficientNet-B4, ResNet50, DenseNet121) against GAN and diffusion-generated imagery. Training dataset: FaceForensics++. Evaluation datasets: CelebDF-v2 and GenImage. Includes JPEG compression simulation at q=90,70,50,30.
